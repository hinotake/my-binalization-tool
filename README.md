# 2値化インタラクティブツール

画像のグレースケール変換および2値化（Binarization）の仕組みを直感的に理解するための教育用ツールです。

## 特徴
- **インタラクティブなヒストグラム**: グラフをクリックすることで、リアルタイムにしきい値を調整し、2値化の結果を確認できます。
- **輝度情報の取得**: 元の画像をクリックすると、その地点の座標と輝度値をカラーチップ形式で表示します。
- **直感的なUI**: しきい値より「白」になる領域と「黒」になる領域を色帯で視覚化しています。
